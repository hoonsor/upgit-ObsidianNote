# 1110804- 提醒新進短期臨工注意事項

## 1. 每日出勤時間
- 0800-1700，如果離開需要協助關燈關門

## 2. 注意事項
- 每個月上班時數上限 150 小時，超過就要請假

## 3. 工作內容
- 協助整理資料
- 接聽電話
- 送公文
- 泡茶、泡咖啡
- 澆花
- 環境清掃
- 碎紙
- 倒垃圾
- Word、Excel 電腦作業事項

## 4. 攜帶文件
- 施打三劑疫苗證明（不然每週需要檢附一次快篩證明）
- 身心障礙證明
- 身份證
- 戶籍謄本或戶口名簿影本
- 郵局帳戶（薪資約定轉帳帳戶）
- 詢問是否有低收入戶證明（健保不用保在本校）
