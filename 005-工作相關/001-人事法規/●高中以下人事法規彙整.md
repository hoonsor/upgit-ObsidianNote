# ●高中以下人事法規彙整

## 1. 法規取代原則
```

●有章節的話
^第(.*?)([章節])(.*)\s
> #h/maroon ==第(\1)(\2)(\3)==^10\n

●法條改H2
(^第.*?條$)
## (\1)

●法條改H2（例外）
本條文有附件 (.*?條)
## (\1)


●項次改H3
([^#\s].*?[^條])\s\s
### (\1)\n

●款次改H4
### ([一二三四五六七八九十])
#### (\1)

●目次改H5
### （
##### （

●換行
(#{3,} )
(\1)\n

●取代只有一項的條文內容（處理中）
條\s\s### \n(.*)\n## 第

```
